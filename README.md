# HoppyDoge

HoppyDoge is a voice controlled HTML5 browser game! Such wow! Very much amaze.

## How it works

+ Help doge to avoid incoming objects by saying "wow".

## Dependencies

+ Firebase
+ Underscore
+ Jquery
+ Web Audio API
+ A Working Microphone
